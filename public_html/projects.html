BrickBreaker, CS 1110
Although simple and ordinary, BrickBreaker is included on my website because nostalgia. I did this with Victoria Beall, my partner for the class. In retrospect, this was a pretty fun assignment, even though I was stuck on one function for a week because I wasn't careful with recursion.

BrickBreaker was a good project because it introduced me to the MVC design pattern and how to effectively use states to represent different modes the game could be in at any given moment.

My first game of BrickBreaker was done in Python, but I will probably
re-implement BrickBreaker in Javascript and link it to this page
for kicks and giggles.

CritterWorld, CS 2112
If you ask me what Spring 2014 tasted like, I'd say it tasted like CritterWorld. I spent eons of time on this project with my partner
Charlie Qian. The last four assignments in CS 2112 culminate cumulate into a giant project known infamously as CritterWorld.

CritterWorld involved parsing & fault injection [spec],
interpreters [spec], working with Java GUIs
& this thing called FXML [spec], and networks [spec].

I also developed some character art and UI design for the game which did not make the final cut because the fire nation attacked. Overall, this project made me stronger as a programmer and as a person, but there are hours upon hours of sleep I fear I'll never be able to quite make up.

PleaseCall, Bloomberg Hackathon Feb 2014
I worked with Junia George, Dashiell Brown, Leon Zaruvinsky, and
Keshav Verma to produce our app titled "PleaseCall".

PleaseCall is an Android app designed to provide flexible functionality as a reminder system for college students who may need to be prompted to call back home. It tracks the number of days spent without calling and/or texting and will send alerts to the student every set number of days.

My role on the team was varied; I did some UI design using XML and worked on some of the database using MySQL. Most of my time was spent studying the Android documentation and learning how to use Git. As this was my first hackathon, I had a lot of learning to do.

We presented our app 24 hours later and were declared winners. Each of us received a Leap Motion, which was an exciting prospect because of how Leap Motion can be used with Unity.
